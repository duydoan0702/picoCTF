## 🧩 Challenge: Obedient Cat

### 📝 Description
Let's start with a simple challenge. Can you read the flag?

> Link: https://play.picoctf.org/practice/challenge/18  
> File: `flag`

### 🛠️ Cách giải

1. Tải file `flag`
2. Dùng lệnh:

```
cat flag
```

3. Flag hiện ra như sau:

```
picoCTF{meow_welcome_to_ctf}
```

### 🏁 Flag

```
picoCTF{meow_welcome_to_ctf}
```
